            # HRMS Frontend
            Minimal React frontend for HRMS demo.

            ## Quick start
            1. `npm install`
2. Start: `npm start`

This app expects the backend APIs under the same origin or to configure proxy in package.json or use environment variables to set API base URL.

Notes: This is a lightweight scaffold to demonstrate UI flows: login/signup, employee CRUD, team CRUD, assignments, and logs.
